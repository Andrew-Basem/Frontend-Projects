
# Fast-Pizza-React

A small app for ordering delicious pizzas


## Demo

https://pizza-ordering0.netlify.app/


## Tech Stack

**Client:** React Router data loading, Redux, Redux Toolkit, thunks, Tailwind CSS




## Installation

To download all packages as node modules
```bash
  npm install 
```

To run on localhost
```bash
  npm run dev 
```
